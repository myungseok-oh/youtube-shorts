# 동물 심리학 연재 플랜

## 시리즈 컨셉

- **포맷**: 1회 = 동물 1가지 행동 → 심리학적 해석 (60초 쇼츠)
- **채널명 후보**: "동물심리 60초" / "반려동물 심리학" / "냥심리"

---

## 연재 구조 (시즌제)

### 시즌 1: 고양이 심리 (6~10화)

| # | 주제 | 핵심 키워드 |
|---|------|-----------|
| 1 | 고양이가 배를 보이는 이유 | 신뢰 신호 |
| 2 | 고양이가 천천히 눈 깜빡이는 이유 | 호감 표현 |
| 3 | 고양이가 물건을 떨어뜨리는 이유 | 관심 끌기 행동 |
| 4 | 고양이가 갑자기 뛰어다니는 이유 | 에너지 방출 |
| 5 | 고양이가 상자에 들어가는 이유 | 안전 욕구 |
| 6 | 고양이가 새벽에 깨우는 이유 | 사냥 본능 |

추가 후보: 골골송, 꾹꾹이, 엉덩이 들기, 화장실 따라오기 등

### 시즌 2: 강아지 심리 (6~10화)

꼬리 흔들기, 하품 따라하기, 발 핥기, 분리불안 등

### 시즌 3: 비교 심리 (고양이 vs 강아지)

같은 상황에서 반응 차이 비교

### 시즌 4+

야생동물, 새, 이색 반려동물로 확장

---

## 1화 대본 구성 (60초 기준)

| 슬라이드 | 내용 | 시간 |
|---------|------|------|
| 인트로 | 후킹 질문 ("고양이가 배를 보여주면 만져도 될까?") | ~5초 |
| 행동 묘사 | 실제 상황 설명 (공감 유도) | ~10초 |
| 심리 해석 | 학술적 근거 + 쉬운 설명 | ~25초 |
| 실전 팁 | 반려인이 어떻게 대응하면 좋은지 | ~15초 |
| 클로징 | 다음 화 예고 + 구독 유도 | ~5초 |

---

## 차별화 포인트

1. **"왜?"에 집중** — 단순 행동 소개가 아니라 진화심리학/동물행동학 근거 제시
2. **반전 요소** — "배를 보여줘도 만지면 안 되는 이유" 같은 예상 뒤집기
3. **시리즈 넘버링** — 썸네일에 #1, #2 표기 → 몰아보기 유도
4. **실전 팁 포함** — 정보 + 실용성 → 저장/공유율 상승

## 시스템 운영 방안

- 새 채널 등록 (ch-0006 등) → `fixed_topic: true` 설정
- 대본 생성: 주제를 `default_topics`에 넣고 Phase A 실행
- 배경 이미지: Gemini로 동물 일러스트/사진 자동 생성 (사람/얼굴 없이 동물만)
- `image_prompt_style`: 동물 중심 밝은 톤, 귀여운 일러스트 or 실사 사진
